I chose to learn web development because...

  • Computer related jobs are growing in the Utah valley.
  • People I know, who work as developers, really love it and they suggested I try it out.
  • I enjoy digital design and Web development is a great way to expand that.
  • It provides variety because it can be applied to many types of business.
  • It seems like it would be a great job to have.

I'm Most Excited to Learn...

HTML 5

HTML 5 and CSS are exciting to learn because they will allow me to control how a website looks and it's fun to experiment with different elements of design. Learning how to put together a framework with HTML and how to use CSS to style a page will be give me the freedom to be more creative. It's been exciting to experiment with HTML and CSS to design the look of a web page. The process can sometimes be frustrating, especially when something isn't working, but it's rewarding when you are able to come up with a design you like and can put it on a web page.
